## Community Distribution Exceptions

### Why these exceptions exist

Open source is not a business model — it is a foundation. The tools and
distributions listed below are part of what made this project possible.
Debian and Ubuntu provide the infrastructure millions of servers run on.
VLC, FFmpeg, and Kyber have demonstrated for decades that open source
software can be technically excellent, widely deployed, and ethically
maintained. This project would not exist without them. These exceptions
are a small way of giving back.
