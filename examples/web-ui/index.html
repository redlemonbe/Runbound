RUNBOUND v0.6.4 AGPL-3.0

High-performance DNS server written in Rust — drop-in replacement for Unbound with a full REST API, built-in blocklist management, XDP zero-copy fast path, and real-time observability.

Features
  • ›XDP zero-copy receive path (AF_XDP)
  • ›REST API — Bearer auth, rate-limited
  • ›Blocklist feeds (hosts / adblock)
  • ›DoT upstreams with TCP+TLS health probe
  • ›HMAC-SHA256 audit chain on all stores
  • ›DNSSEC validation (optional)

API Reference▾
Auth: Authorization: Bearer <key> — key in /etc/runbound/api.key
Rate limit: 30 req/s per IP, burst 60  ·  Max body: 64 KiB  ·  Port: 8080
Health & Stats
GET/healthNo auth — liveness probe — {status, version, uptime_secs}
GET/api/statsQPS, latency percentiles (p50/p95/p99), counters, DNSSEC stats
GET/api/systemVersion, CPU, memory, XDP mode, upstream health, prefetch flag
GET/api/logsLast 100 query log entries (requires verbosity ≥ 1)
DNS Entries
GET/api/dnsList all local records
POST/api/dns{name, type, value, ttl} — A AAAA CNAME TXT MX SRV PTR SSHFP TLSA
DEL/api/dns/:idRemove by UUID
Blacklist
GET/api/blacklistList blocked domains
POST/api/blacklist{domain, action: "nxdomain|refuse"}
DEL/api/blacklist/:idRemove by UUID
Feeds
GET/api/feeds/presetsBuilt-in preset blocklists
GET/api/feedsList active feeds with stats
POST/api/feeds{name, url, format: "hosts|adblock", action, enabled}
POST/api/feeds/:id/updateForce fetch for one feed
POST/api/feeds/updateForce update all feeds
DEL/api/feeds/:idRemove feed
Upstreams
GET/api/upstreams/presets9 built-in presets (Cloudflare, Google, Quad9, OpenDNS)
GET/api/upstreamsList with health, latency, DNSSEC flag (v0.6.5+), history (v0.6.5+)
POST/api/upstreams{addr, protocol: "udp|dot", port, name} — no loopback/link-local
PATCH/api/upstreams/:idRename — {name} — v0.6.5+
DEL/api/upstreams/:idRemove — 409 LAST_UPSTREAM if last
Cache & Control
POST/api/cache/flushFlush cache — 429 FLUSH_COOLDOWN within 60 s
GET/api/cache/statsHit/miss/eviction counters — v0.6.5+
POST/api/reloadReload config (equivalent to SIGHUP)
GET/api/sync/slavesList connected slave nodes with sync status

Built with Rust · Tokio · Hickory DNS · axum · rustls
